Customization settingsedit

In addition to adjusting the event threshold, users can determine whether an event should trigger after a certain number of minutes. This can be beneficial in cases where it is known that an event often opens and closes quickly. By delaying the trigger until the event remains open for more than X minutes, unnecessary alerts can be reduced. You can also mute notifications of opened events. Changing the event threshold parameters will only apply to newly created events.
